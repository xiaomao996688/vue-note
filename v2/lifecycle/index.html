# 前言

在 Vue 中, new Vue 会创建一个 Vue 的实例，在这个过程当中，这个实例会经历从创建到被销毁。在创建到销毁的整个过程称为生命周期。生命周期的不同阶段对应了 Vue 实例调用不同的生命周期函数。
